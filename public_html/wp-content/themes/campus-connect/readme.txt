== Description ==

Campus Connect is a modern block based WordPress theme built for educational community initiatives and campus projects. It features clean, intuitive layouts, customizable patterns, and flexible templates powered entirely by Full Site Editing. Designed for clarity and adaptability, it makes it easy to create a dynamic, engaging online presence for any academic community.
